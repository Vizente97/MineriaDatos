Análisis correlacional de datos (ACD ó CDA)

¿Qué es el Análisis correlacional de datos?

El ACD (CDA) es útil para reducir el número de variables, de un espacio de alta dimensión a uno de menor número de dimensiones. Esto se logra a través de la identificación de variables significativas. Esta identificación de correlaciones se utiliza para determinar el grado de similitud (relevancia/irrelevancia) de los valores de dos variables numéricas.

Existe correlación entre 2 variables (X,Y) si al aumentar los valores de X también los hacen de Y, o viceversa.

La reducción consiste en lo siguientes 2 puntos:

  • Que a partir de un conjunto de variables originales: X1, X2, X3,..., Xn
  • Se obtiene otro subconjunto de variables relevantes : X1, X2, X3,..., Xm, donde m < n.
