Análisis de componentes principales (PCA)

¿Qué es PCA?

El análisis de componentes principales (ACP o PCA, Principal Component Analysis) es un algoritmo matemático para reducir la cantidad de variables de conjuntos de datos, mientras se conserva la mayor cantidad de información posible.

Estos componentes principales (vectores o factores) son combinaciones lineales, no correlacionadas entre sí, que retienen la mayor cantidad de varianza. El algoritmo trabaja con operaciones matriciales y álgebra lineal (multiplicación de matrices, descomposición de matrices, vectores propios, valores propios) y estadísticas (estandarización, varianza, covarianza).

El cálculo de PCA implica los siguientes 5 pasos:

  1. Se hace una estandarización de los datos.
  2. A partir de los datos estandarizados, se calcula una matriz de covarianzas o correlaciones.
  3. Se calculan los componentes (eigen-vectores) y la varianza (eigen-valores).
  4. Se decide el número de componentes principales.
    • Se calcula el porcentaje de relevancia, es decir, entre el 75 y 90% de varianza total.
    • Se identifica mediante una gráfica el grupo de componentes con mayor varianza.
    • *Se elige las dimensiones cuya varianza sea mayor a 1.
  5. Se examina la proporción de relevancias –cargas–
