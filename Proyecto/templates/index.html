Presentación

¿Qué es la mineria de datos?

Es la exploración y análisis inteligente de grandes cantidades de datos para descubrir reglas y patrones significativos a partir de un conjunto de datos, adicionalmente se puede ver como el proceso computacional para la exploración y análisis inteligente de datos como apoyo para el proceso de la toma de decisiones

Hay aspectos importantes los cuales han ayudado al crecimiento y desarrollo de la minería de datos, como por ejemplo:

  • La creciente disponibilidad de información en Internet.
  • La presión competitiva del mercado en una economía globalizada.
  • El desarrollo de herramientas libres y comerciales para llevar a cabo la minería de datos.

Se tienen áreas en las que puede ser aplicada:

  • Finanzas.
  • Análisis de mercado.
  • Procesos industriales.
  • Medicina.
  • Biología.
Objetivo del proyecto

Este es el proyecto para la clase de Minería de Datos para el semestre 2021-2. El objetivo de este proyecto es mostrar algunos Algoritmos utilizados en esta materia asi como una implementación que se podrá usar en linea.

Algoritmos implementados:

  • EDA.
  • Selección de Características.
  • Métricas.
  • PCA.
  • Cluster Particional (K-Means).
  • Reglas de asociación (Apriori).
